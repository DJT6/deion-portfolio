Multi-Tier Healthcare Appointment Application

Built a cloud-based healthcare appointment booking system on AWS using a multi-tier architecture (EC2, DynamoDB, Route 53). Patients can schedule, cancel, and manage appointments through a secure web frontend with a custom domain and SSL. Implemented Nginx reverse proxy, CloudWatch monitoring, and IAM least-privilege policies, demonstrating scalability, cost optimization, and real-world cloud deployment practices.

Cloud Game Leaderboard Real-Time Multiplayer Stats

Built a real-time multiplayer leaderboard system on AWS using a fully serverless architecture (Lambda, API Gateway, DynamoDB). Enabled players to submit scores, view global rankings, and track top performers through a Pitt-themed web frontend. Real-Time Serverless Contact Form

This project is a fully serverless contact form built on AWS. User submissions are stored in DynamoDB, automatically exported to S3 via DynamoDB Streams, and visualized in QuickSight in near real-time (~15 min). It leverages Lambda, API Gateway, SNS notifications, and CloudWatch metrics, demonstrating a scalable, event-driven AWS architecture with a polished, responsive front-end.
