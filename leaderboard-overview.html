Cloud Game Leaderboard

A real-time, cloud-hosted leaderboard for multiplayer games demonstrating event-driven processing, scalable architecture, and secure AWS deployment.

Project Overview

The Cloud Game Leaderboard is a real-time multiplayer scoring system inspired by Panther games at the University of Pittsburgh. It allows players to submit scores, track global rankings, and watch the leaderboard update instantly as new results come in. Built on a fully serverless AWS architecture, the system scales automatically with traffic and requires no server management. A Pitt-themed web frontend provides an easy way for players to submit scores and follow the top performers, creating a live, competitive experience. A project like this can be applied to any interactive competition or event where real-time tracking matters such as campus gaming tournaments, classroom challenges, hackathons, or even sports trivia nights. It demonstrates how cloud technology can turn a simple game into a dynamic shared experience where players see results update live and feel connected through competition..
