Project Overview

The Healthcare Appointment Booking System is a cloud-based web app where patients can book, cancel, and manage appointments without conflicts or double-bookings. The frontend runs on S3 with CloudFront, while the backend is deployed on EC2 instances in an Auto Scaling Group behind an Application Load Balancer with HTTPS. Data is stored in DynamoDB with IAM least-privilege access and encryption, and the VPC is segmented into public and private subnets to reflect HIPAA-aligned security practices. CloudWatch and AWS Budgets provide monitoring and cost control, and SNS sends appointment confirmations and reminders. The system demonstrates how an EC2-based workload can be built with healthcare security principles while maintaining scalability and reliability.
