Project Overview

The Healthcare Appointment Booking System is a cloud-based web app where patients can book and manage appointments online. The frontend is served through Nginx with a custom Route 53 domain and HTTPS using Let’s Encrypt. The backend is a Node.js/Express API deployed on an EC2 instance inside a custom VPC, with DynamoDB storing appointment data under IAM least-privilege access. Nginx is configured as a reverse proxy to connect the API with the frontend securely. Route 53 handles DNS, while CloudWatch and AWS Budgets provide monitoring and cost control. The system demonstrates how to deploy a multi-tier application on AWS with security, scalability, and cost optimization in mind.
