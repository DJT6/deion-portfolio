AI Log Detection and Response System

This project focuses on building a cost-efficient, serverless detection pipeline capable of identifying suspicious activity in real time. Logs are streamed, processed, and analyzed with Amazon SageMaker to detect anomalies and trigger alerts or automated blocking. It’s designed for on-demand processing to minimize costs while still demonstrating strong operational automation and security monitoring. The system provides a clear example of integrating ML-driven detection with a modern cloud-native architecture.

Architecture: S3 (log storage) • Kinesis Firehose (streaming) • Lambda (preprocessing) • SageMaker (anomaly detection) • SNS (alerts) • WAF (response) • CloudWatch (monitoring) • GitHub Actions + Terraform

Cloud-Based Appointment Management System

A production-ready appointment platform built with a secure and scalable AWS stack. It includes authentication, monitoring, cost control, and automated deployments through CI/CD. The design focuses on high availability, zero idle cost, and fast recovery from common failure scenarios. This project reflects strong cloud infrastructure practices suitable for real-world use cases.

Architecture: S3 + CloudFront (HTTPS) • API Gateway + Lambda (Node.js) • DynamoDB (conditional writes) • Cognito (auth) • CloudWatch (logs, alarms) • WAF + KMS • GitHub Actions + Terraform

Cloud Game Leaderboard

A real-time serverless leaderboard that allows players to submit scores and view rankings through low-latency APIs. It integrates basic abuse detection and traffic control, ensuring stable performance without increasing costs. The architecture is optimized for scalability, cost efficiency, and simplicity, making it ideal for showcasing event-driven design and operational monitoring.

Architecture: S3 (static frontend) • API Gateway (REST + WebSocket) • Lambda (Node.js) • DynamoDB (top-N leaderboard with GSI) • CloudWatch (monitoring & alarms) • WAF (rate limiting) • GitHub Actions + Terraform

Serverless Contact Form Application

A lightweight contact form application designed to be simple, reliable, and cost-free to operate. It uses serverless components to process submissions, block spam, and automate data cleanup. Notifications are sent in real time, and the architecture requires no manual management, making it ideal for personal websites or small-scale business deployments.

Architecture: API Gateway → Lambda → DynamoDB (TTL cleanup) → S3 + SNS • WAF • CloudWatch • GitHub Actions + Terraform
